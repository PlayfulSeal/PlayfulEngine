# Playful Engine
## Version 0.2.0
---

### Features
- Sprites, Sprite Sheets and Sprite Rendering
- **ECS** (**E**nity **C**omponent **S**ystem)
- Scenes and Scene Management
- Scripting with **GOScript** (**G**ame **O**bject **Script**)
- 2D Camera
- Basic UI w/ Interactable UI elements
---

### Roadmap
- UI Buttons, Text Fields
- Asset Manager
- Animated Sprites
- Tiles & Tilesheets
- Tilemaps
- Levels
- Triggers
- Audio Mixing (BGM, Voice, SoundFX)
---

### How to install

Download "PlayfulEngine - (VERSION).dll" and reference it within your project.
